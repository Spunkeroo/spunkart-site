DNS Record Types Explained

A
Address Record

Maps a domain name to an IPv4 address (e.g. 93.184.216.34). The most fundamental DNS record type.

AAAA
IPv6 Address Record

Maps a domain name to an IPv6 address. Same as A record but for the newer 128-bit IP format.

MX
Mail Exchange

Specifies the mail server responsible for receiving email. Includes a priority value (lower = preferred).

NS
Name Server

Indicates which DNS servers are authoritative for the domain. Usually set by your domain registrar.

TXT
Text Record

Holds arbitrary text data. Commonly used for SPF, DKIM, domain verification, and security policies.

CNAME
Canonical Name

Creates an alias from one domain to another. The target domain's A record is used for resolution.

TTL (Time To Live) = how many seconds DNS resolvers should cache a record before refreshing it. Lower TTL = faster propagation when changed. Typical values: 300 (5 min) to 86400 (24 hr).
