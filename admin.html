Cloudflare Setup Guide

Add Cloudflare to all your domains for free CDN, DDoS protection, and speed boost.

Step 1: Sign up at cloudflare.com (free plan)
Step 2: Add each domain and change nameservers at your registrar
Step 3: Enable these settings on each domain:
Speed Settings (Free Plan)
Enable Auto Minify (HTML, CSS, JS)
Enable Brotli Compression
Enable Early Hints (103)
Enable HTTP/2 & HTTP/3 (QUIC)
Enable Always Use HTTPS
Enable Automatic HTTPS Rewrites
Enable Browser Cache TTL: 1 month
Enable Rocket Loader (test first)
Security Settings
Enable Bot Fight Mode
Enable Browser Integrity Check
Set Security Level: Medium
Set SSL/TLS: Full (Strict)
Enable HSTS (max-age 6 months)
Set Min TLS Version: 1.2
Enable Opportunistic Encryption
