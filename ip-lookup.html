How to Use the IP Address Lookup Tool

Enter any public IP address into the search box and click "Lookup" to instantly see detailed geolocation information. You can also click "Show My IP Address" to discover your own public-facing IP and its associated data.

What Information Does the IP Lookup Show?

  • IP Address — The IPv4 or IPv6 address being queried
  • Country & City — The approximate geographic location of the IP
  • Region — The state, province, or administrative region
  • ISP / Organization — The internet service provider or hosting company
  • Timezone — The local timezone associated with the IP's location
  • Coordinates — Latitude and longitude with a link to Google Maps
  • ASN — The Autonomous System Number of the network

Frequently Asked Questions

Is this tool free? Yes, the IP Address Lookup tool is completely free with no signup required.

Is the location exact? IP geolocation provides an approximate location, typically accurate to the city level. It does not reveal a precise street address.

Can I look up IPv6 addresses? Yes, both IPv4 and IPv6 addresses are supported.
